# 🎰  經典拉霸機（Slot Machine）

## 📁 結構
```
slot_machine_web/
├── app.py                ← Flask 主伺服器
├── static/
│   └── style.css         ←（選擇性）樣式
└── templates/
    └── index.html        ← 前端介面
```


```bash
python slot_app.py
```
啟動後開啟網頁：http://127.0.0.1:5000  
可進行拉霸機遊戲
