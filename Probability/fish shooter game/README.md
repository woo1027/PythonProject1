# 🎣 魚機（Fish Shooter Game）
不同魚有不同HP與擊殺機率，子彈花費固定。

每次射擊消耗一發子彈（成本1）。

若隨機機率小於魚的出現機率 (prob)，則算擊中，賺取魚的獎勵 (reward) 減去子彈成本。

否則沒擊中，只損失子彈成本。

## 📁 結構
```
fish_shooter_game/
├── fish_app.py                ← Flask 主伺服器
└── templates/
    └── index.html        ← 前端介面
```


```bash
python fish_app.py
```
啟動後開啟網頁：http://127.0.0.1:5000  
可進行魚機

## 玩家體驗化
- 玩家決定要打哪種魚:大魚高報酬但低命中;小魚易死但低收益
- 動態調整魚群數量與出現頻率
